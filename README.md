# 那年夏天的樱花雨

一个由 Ren'Py 引擎驱动的校园青春恋爱互动视觉小说。

## 故事简介

大二学生林墨在春日的樱花道上偶然撞见了中文系文学社社长苏晴。从那一刻起，一段青涩而温暖的校园恋爱悄然萌芽。图书馆里的低语、旧书店里的午后、樱花树下的心动……每一步选择都将决定这段感情的走向。

## 游戏特色

- 约 5000 字完整剧情
- 4 个关键选择点，影响故事走向
- 3 个结局：好结局「樱花雨下」、普通结局「各自远方」、坏结局「错过」
- 纯文字叙述，无需立绘素材即可运行

## 运行方式

本项目需要 [Ren'Py](https://www.renpy.org/) 引擎运行。

1. 安装 Ren'Py SDK
2. 将本项目导入 Ren'Py 项目管理器
3. 点击「启动项目」即可开始游戏

## 项目结构

```
game/
├── script.rpy          # 主剧本（完整故事与选择分支）
├── characters.rpy      # 角色定义与好感度变量
├── backgrounds.rpy     # 纯色背景定义
├── options.rpy         # 游戏配置
├── gui.rpy             # GUI 配色与样式
├── screens.rpy         # 界面屏幕定义
├── SourceHanSansLite.ttf  # 思源黑体字体
├── gui/                # GUI 素材资源
└── tl/                 # 翻译文件
```
